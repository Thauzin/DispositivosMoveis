# 📱 **Dispositivos Móveis** 

## 📌 Como acessar as atividades? ##
Para encontrar as atividades acesse: exemplos_dart/bin/[nome da atividade] <br>
Obs: O nome da atividade está identificado em cada descrição. 

## 📝 Atividade 1 
A primeira atividade foi para conhecer um pouco da linguagem dart. <br>
No código "exemplos.dart", você pode observar a utilização de comandos básicos como: imprimir na tela, 
declarar os diferentes tipos de dados (int, double, String, etc) e por fim criar uma classe.
